## Running Locally

This application requires Node.js v16.13+.

```bash
git clone https://github.com/leerob/leerob.io.git
cd leerob.io

```

Create a `.env` file similar to [`.env.example`](https://github.com/leerob/leerob.io/blob/main/.env.example).
